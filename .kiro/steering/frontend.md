# Frontend Guidelines — AGON

## Theme: Futuristic (high priority)
The AGON web UI must look **as striking and futuristic as possible** — this is a
hackathon, visual wow-factor matters for judging and the demo video.

Direction:
- Dark, high-contrast "command center / arena" aesthetic.
- Neon / glow accents (cyan, magenta, electric blue), subtle gradients.
- Glassmorphism panels, animated backgrounds (grid/particles), smooth motion.
- Monospace/techno typography for data; clean sans for body.
- Live, dynamic feel: animated leaderboard, real-time match ticker, pulsing
  "Human or AI?" market odds.
- Micro-interactions and tasteful motion (Framer Motion) over static pages.

Stack (per docs/MVP_SCOPE.md): Next.js + TypeScript + Tailwind + viem/wagmi +
RainbowKit. Consider Framer Motion for animation.

Keep it performant and accessible despite the heavy visuals.
